Google Docs

When I write text for my websites I write them first in Google Docs. My goal when using this is to make a draft of the text and then edit it till I'm satisfied with it. I then insert it into the website. Google Docs is a free text document creator and editor provided by google. Google docs allows for me to change to document in ways such as font, color, and size. It has a very high level of convienience. One of the most convienient things is that it automatically saves any progress that happens to it, so you don't have to worry about losing your progress. Another thing is that it saves to the cloud and can therefore be accessed anywhere at any time.
